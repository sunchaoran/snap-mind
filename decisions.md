# Phase 1: Decisions

## Setup Parameters
- **Project**: SnapMind (Dev Console / Screenshot Upload Page)
- **Niche**: Memory capture / automated knowledge base engine
- **Selected Director**: Christopher Nolan
- **Selected Film**: Inception (盗梦空间)

## Research & Cinematic DNA
**Research Source**: 
- `directors-200.md` -> Christopher Nolan (Time manipulation, IMAX scale, rational structure, monochrome+blue)
- *Inception* (2010): The aesthetic of nested realities, folding architecture, paradox, zero gravity corridors, physical manipulation of memory/dreams. Clinical reality vs. deep subconscious.

**Interpretation Note**:
Instead of a generic SaaS upload page, the SnapMind dev portal should feel like an extraction terminal or a dream machine. The act of uploading a screenshot is passing memory from the conscious (surface level) into the subconscious (deep processing LLM vault). It should be tactile, monumental, and mathematically precise. 

## Uniqueness Audit
- **Previous-work audit**: (No prior cinematic-ui projects exist for this user in this workspace).
- **Shell-ban list**: 
  - NO centered soft gradient backgrounds.
  - NO standard white dropping shadows on "upload cards."
  - NO bubbly, rounded "drop files here" dotted borders.
  - NO generic SaaS empty states.

## Architecture
- **Primary composition family**: Nested Monoliths / Depth Corridors
  *The interface isn't just flat cards; it has Z-index depth. Uploading a file physically "pushes" it into the page (down a layer into the dream state).*
- **Interaction Posture**: Rational precision. Everything clicks into place mechanically (like a spinning top or turning gears).
- **Page List**:
  1. `Upload Terminal` (The surface reality: dragging a file in)
  2. `Extraction State` (The descent: LLM analyzing in progress, visual representation of parsing JSON/images)
  3. `Vault Result` (The bottom of the dream: final parsed markdown/mind-map preview)

## Image Placeholders
- Real uploaded screenshots will be used. Where empty states exist, we will use clinical, wireframe-like mathematical diagrams (resembling paradoxical architecture or maze blueprints) rather than friendly vector art.
