# Changelog

## [0.2.0] — 2025-04-08 · 盗梦空间 (Inception)

> 记忆萃取器的第一次重大视觉与引擎升级。
> 灵感来源：Christopher Nolan《Inception》(2010)

### ⚙️ 引擎重构

- **LLM 执行核心迁移**：从 OpenRouter 全面切换至 MiniMax-M2.7，采用中国区端点 (`api.minimaxi.com/v1`)，大幅提升本地化截图萃取的推理速度与稳定性。
- **泛型 Provider 架构**：取消硬绑定的 API 配置，采用 `LLM_API_KEY` + `LLM_BASE_URL` 双变量设计，支持未来多端供应商快速切换，无需改动源码。
- **JSON 解析鲁棒性增强**：重写底层 `parseLLMJson` 启发式抽取算法，新增激进尾部截断策略，彻底杜绝模型偶发性输出非 JSON 文本导致的管线崩溃。

### 🎬 Cinematic UI · 盗梦空间视觉系统

- **深邃黑曜石基底**：抛弃原有 SaaS 扁平风格，引入 `--void: #06080f` 至 `--surface-glass` 五层纵深色阶体系，搭配 Penrose 几何网格大气层（CSS mask-image 实现）。
- **梦境双色调**：钢蓝色 (`--steel: #4a6fa5`) 代表现实层冷峻理性，琥珀金 (`--amber: #c4943a`) 代表酒店层温暖记忆，两色交织贯穿整套交互。
- **毛玻璃材质面板**：上传区与结果面板均采用 `backdrop-filter: blur(20px)` 重度 Glassmorphism 处理，模拟梦境中透明却有阻隔的边缘质感。

### 🌀 交互动效

- **偏心展现动画 (Off-center Reveal)**：初始状态上传面板居中孤立；选择截图后，面板以 0.8s 电影级缓冲曲线 (`cubic-bezier(0.22, 1, 0.36, 1)`) 平滑滑向左侧，右侧梦境监控面板从零宽度横向拉开呈现。
- **图腾验证系统 (The Totem)**：处理进行中的时间线节点内置永续旋转的琥珀色半环图腾动效，暗示当前所处的梦境下潜深度。节点完成后转为翠绿 ✓，失败则标红 ✗。
- **记忆保险箱浮升 (The Safe)**：提取结果面板采用 `panel-rise` 动画（`translateY(16px) → 0`，`scale(0.98) → 1`），从深渊中缓缓浮起。

### 📐 响应式布局

- **桌面端双列 Flex 布局**：左侧固定 460px 上传/历史区，右侧自适应宽度状态/结果区，中间 4rem 间距由 `gap` 动画驱动。
- **移动端垂直降级**：窄屏自动退化为单列纵向堆叠，时间线节点尺寸与字号同步缩小。
- **Reduced Motion 适配**：`prefers-reduced-motion` 媒体查询下关闭所有动画与过渡。
